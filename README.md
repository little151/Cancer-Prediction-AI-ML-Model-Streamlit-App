# Lung Cancer Prediction on Image Data

## Overview
This repository contains the code for lung cancer prediction on image data. The project utilizes machine learning techniques to predict the likelihood of lung cancer based on medical images.

## Dataset
The dataset used in this project can be found on Kaggle. You can download the dataset from the following link:
[Download Lung Cancer Dataset](https://www.kaggle.com/code/adityamahimkar/lung-cancer-prediction-on-image-data)
